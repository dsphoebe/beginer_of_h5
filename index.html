什么是HTML

  • HyperText Markup Language
  • 超文本标记语言
  • 1989年 欧洲原核子研究中心(CERN)的Tim Berners Lee基于SGML发明

HTML的作者

  • Tim Berners Lee
    • 物理学家
    • 计算机科学家
    • 发明了HTML，HTTP
    • WWW发明人
    • W3C创建人
    • 骑士爵位

HTML诞生

  • 1980 Tim信息管理系统
    • 易于编辑
    • 易于分享
  • 1989 HTML诞生
  • 1990 Tim开发了浏览器还有相关软件

HTML1.0

  • 1991 Tim发布HTML1.0草案
    • 只有18个标签
    • 没有img标签
    • HTML初版
    • Mosaic1.0兼容

Mosaic

  • 国际超级计算机应用中心(NCSA)
  • 对网络的发展产生很大的影响
  • 后来的IE也是用了它的代码
  • 作者：马克.安德森
    • 离开NCSA后 重写了另一个红极一时的浏览器Netscape
    • Nestcape后来由于Micrsoft的windows绑定IE销售 最后死了

W3C

  • 1994 Tim创立W3C
    • HTML的流行，兼容性问题
    • 制定了CSS，XML等一系列深远影响的标准
    • 推荐标准 不强制
  • 这一年，跟Tim同在CERN的同事Håkon Wium Lie提出了CSS

HTML2.0

  • 1995 HTML2.0正式版
    • Document type definition DTD 文档类型定义
    • 浏览器或者解析器用这种文档类型来渲染HTML页面
    • 看看页面
  • 1996年 W3C发布CSS1

HTML3.2

  • 1997 HTML3.2
    • 加入了font，u，address，style，script等标签
    • 增强了标签属性
    • 引入了颜色，可以给字体设置颜色了
    • 看看页面

HTML4

  • 1998 HTML4
  • 同年 W3C发布CSS2

XHTML1.0

  • 2000, 2002 XHTML1.0
    • HTML vs XHTML vs XML

WHATWG

  • 2004 WHATWG
    • Mozzil, Opera, Apple
    • 创建

W3C + WHATWG = HTML5

  • 2007 W3C介入制定HTML5标准
