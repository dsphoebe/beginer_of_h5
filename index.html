什么是HTML

  • HyperText Markup Language
  • 超文本标记语言
  • 1989年 欧洲原核子研究中心(CERN)的Tim Berners Lee基于SGML发明

HTML的作者

  • Tim Berners Lee
    • 物理学家
    • 计算机科学家
    • 发明了HTML，HTTP
    • WWW发明人
    • W3C创建人
    • 骑士爵位

HTML诞生

  • 1980 Tim信息管理系统
    • 易于编辑
    • 易于分享
  • 1989 HTML诞生
  • 1990 Tim开发了浏览器worldwideweb，后来改名Nexus

HTML1.0

  • 1991 Tim发布HTML1.0草案
    • 只有18个标签
    • 没有img标签
    • HTML初版
    • Mosaic1.0兼容

Mosaic

  • 国际超级计算机应用中心(NCSA)
  • 对网络的发展产生很大的影响
  • 后来的IE也是用了它的代码
  • 开发者马克.安德森
    • 离开NCSA后 重写了另一个红极一时的浏览器Netscape
    • Nestcape后来由于Micrsoft的windows绑定IE销售 最后死了

W3C

  • 1994 Tim创立W3C
    • HTML的流行，兼容性问题
    • 制定了CSS，XML等一系列深远影响的标准
    • 推荐标准 不强制
  • 这一年，跟Tim同在CERN的同事挪威人Håkon Wium Lie提出了CSS

HTML2.0

  • 1995 HTML2.0正式版
    • Document type definition DTD 文档类型定义
    • 浏览器或者解析器用这种文档类型来渲染HTML页面
    • 看看页面
  • 1996年 W3C发布CSS1

HTML3.2

  • 1997 HTML3.2
    • 加入了font，u，center等样式标签，style，script等head标签
    • applet java标签
    • 引入了颜色，可以给字体设置颜色了
    • 看看页面

HTML4

  • 1998 HTML4
    • 移除了font, u, center等样式标签，样式通过CSS来实现了
    • 移除了applet java标签，改用object标签
    • id，class，style属性
    • 表单也加了好几个属性 disabled，readonly等
  • 同年 W3C发布CSS2

XHTML1.0

  • 2002 XHTML1.0
    • HTML4的XML版本
    • XML Extensible Markup Language 可扩展标记语言
    • XML可以完全实现自定义标签
    • W3C是希望开发者能通过XHTML1.0从HTML过渡到XML
    • 有3个DTD

XHTML1.0 vs HTML4

  • 语法更严格：
    • 标签必须小写
    • 必须封口
    • 属性必须用双引号引起来
  • 减小浏览器的负担

WHATWG

  • 2004 WHATWG
    • Web Hypertext Application Technology Working Group
    • Mozilla, Opera向W3C提交web application草案
    • 对web向下兼容的情况下推出了一些新的功能
    • Mozilla, Opera, Apple组建

W3C + WHATWG = HTML5

  • 2007 W3C采纳HTML5作为标准，并成立新的HTML小组继续制定标准

乔大帮主发文

  • 2010 乔大帮主发文 关于Flash的思考
  • HTML5 vs Flash
  • 2011 Adobe放弃Flash

HTML5

  • 2014 W3C发布HTML5正式版
    • CSS3 ＋ JS ＋ HTML
    • web application 1.0 ＋ web form 2.0

HTML5 vs HTML4

  • HTML5标准并不只是HTML
  • HTML: XHTML追求的语义化
  • CSS3 + JS: 只有Flash能实现的复杂动画效果
  • JS: 本地缓存，定位等

总结

  • 1989 HTML1, 1995 HTML2, 1996 CSS1 + JavaScript
  • 1997 HTML3.2, 1998 HTML4 + CSS2
  • 2002 XTML1
  • 2014 HTML5 (CSS3 + HTML + JavaScript)
